Este repositorio aloja el código fuente de la evaluación parcial 2 del curso Desarrollo Full Stack 2, el cual corresponde a una aplicación de página única (*single page application*) de una tienda de e-commerce de venta de películas llamada **MoviE-Store**.

Este proyecto fue desarrollado con Reactjs, Typescript y Vite.

Para poder utilizar el proyecto, debe tener instalado node.js en su dispositivo. Luego de haberlo instalado, debe abrir una terminal y, en primer lugar, ejecutar el siguiente comando para instalar las dependencias de la aplicación:

```bash
npm install
```

Después de haber instalado las dependencias, podrá ejecutar la aplicación a través del siguiente comando:

```bash
npm run dev
```

Al ejecutar este comando, Vite creará un servidor web local y en la terminal le enviará la URL junto con el puerto para acceder a la aplicación, la cual suele ser `http://localhost:5173`.
